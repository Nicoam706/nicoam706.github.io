Descripción del Proyecto

Estudio de un portal mágico en tiempo real, desarrollado con Niagara en Unreal Engine 5, explorando materiales personalizados, emisores basados en mesh y distorsión para lograr un efecto dinámico, claro y con un estilo mágico adecuado para gameplay. El proyecto profundiza en la legibilidad visual de la forma y el movimiento, implementando módulos e ideas experimentales para mantener siluetas claras y un comportamiento visual coherente.

Características Técnicas

• Materiales personalizados y emisores basados en mesh
• Distorsión dinámica para efecto mágico
• Enfoque en legibilidad visual y siluetas claras
• Módulos experimentales de Niagara
• Comportamiento visual coherente para gameplay

Descripción del Proyecto

VFX de impacto de meteorito en tiempo real, creado en Unreal Engine 5 con Niagara. El efecto combina fuego, humo, chispas, escombros y una explosión final, enfocándose en timing, claridad y sensación de peso para que funcione de forma convincente en un entorno de juego. Se optimizó el sistema usando partículas GPU donde no era necesaria colisión y materiales simples para mantener rendimiento y legibilidad visual.

Características Técnicas

• Múltiples sistemas de partículas sincronizados (fuego, humo, chispas, escombros)
• Explosión final con impacto visual
• Partículas GPU para optimización
• Materiales simples para legibilidad visual
• Timing y peso convincentes para gameplay
